# AuthFlow 🔐

> A full-stack authentication and user management system built with the MERN stack featuring secure JWT workflows, protected routing, role-based access control, and scalable backend architecture.

Designed to explore real-world authentication systems, session handling, API security, and structured frontend-backend integration.

---

## Features

- JWT-based authentication and authorization
- Secure login and registration workflows
- Protected frontend and backend routes
- User session and token management
- Admin dashboard with user management controls
- CRUD operations for user administration
- Role-based access workflows
- RESTful API integration between frontend and backend

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React |
| Backend | Node.js, Express.js |
| Database | MongoDB |
| Authentication | JWT |
| Styling | CSS / TailwindCSS |
